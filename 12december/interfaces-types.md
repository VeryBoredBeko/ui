# Тақырып: Интерфейстердің түрлері және олардың ерекшеліктері

## Мақсаты:
Студенттерге әртүрлі интерфейстерді тануды, олардың ерекшеліктерін анықтауды және салыстыру арқылы талдауды үйрету.

## Тапсырма:

### 1. Теориялық бөлім
1. Интернеттен немесе оқулықтан келесі интерфейстер туралы ақпарат жинаңыз:
   - **CLI (Command Line Interface)** – пәрмен жолымен басқару интерфейсі.
   - **GUI (Graphical User Interface)** – графикалық интерфейс.
   - **TUI (Text-based User Interface)** – мәтіндік интерфейс.
   - **VUI (Voice User Interface)** – дауыстық интерфейс.
2. Әр интерфейстің негізгі ерекшеліктерін қысқаша жазып шығыңыз:
   - Қолданушыға ыңғайлылығы
   - Пайдалану сценарийлері
   - Артықшылықтары мен кемшіліктері

### 2. Практикалық бөлім
1. Үш түрлі интерфейсті таңдап, олардың скриншоттарын немесе суреттерін табыңыз.
2. Әр интерфейстің элементтерін бөлшектеп сипаттаңыз (мысалы, батырмалар, мәзірлер, пәрмен жолы, дауыс командасы).
3. Қысқаша салыстыру жасаңыз:
   - Қай интерфейс қай жағдайда тиімді?
   - Қайсысы студент үшін ыңғайлырақ?
   - Қандай жақсарту ұсыныстары бар?

### 3. Есеп беру
- Барлық жазбаларды және скриншоттарды бір құжатқа жинаңыз.
- Құжатты PDF форматына ауыстырып, тапсырманы тапсырыңыз.
