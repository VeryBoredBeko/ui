# Интерфейстердің түрлері және олардың ерекшеліктері

## Теориялық бөлім
Интерфейс — қолданушы мен жүйе арасындағы өзара әрекеттесу құралы. Интерфейстер бірнеше түрге бөлінеді:

- **CLI (Command Line Interface)** – пәрмен жолымен басқару.  
- **GUI (Graphical User Interface)** – графикалық элементтер арқылы басқару.  
- **TUI (Text-based User Interface)** – мәтіндік интерфейс, визуалды элементтері бар.  
- **VUI (Voice User Interface)** – дауыстық командалар арқылы басқару.  

**Ерекшеліктері:** қолданылу жағдайы, ыңғайлылық, күрделілік деңгейі.

## Тапсырма
- **Файл:** `interfaces-types.md`  
- **Нұсқаулық:**  
  1. Әр интерфейстің скриншоттарын немесе суреттерін табу.  
  2. Скриншоттарға қысқаша сипаттама жазу (ерекшеліктері, қолдану жағдайы).  
  3. Барлық скриншоттарды бір PDF құжатында жинап тапсыру.  

> **Ескерту:** Бұл тақырып бойынша тапсырмалар тек `interfaces-types.md` файлына орналасқан.

# Түс және оның интерфейстегі рөлі

## Теориялық бөлім
Түс қолданушының қабылдауын жеңілдетеді, назар аудартады және интерфейс эстетикасын арттырады. Дұрыс түстер ақпаратты ұйымдастыруға, маңызды элементтерді бөліп көрсетуге, пайдаланушы эмоциясын қалыптастыруға көмектеседі.  

## Тапсырма
- **Кітап:** `Niederst_Robbins_Jennifer_Learning_Web_Design-323-374.pdf`  
- **Exercise:** 13.1 – 13.8  
- **Нұсқаулық:**  
  1. Тапсырмаларды орындаңыз (скриншоттармен).  
  2. Ағылшын тіліндегі мәтінді аудару қажет болса, ИИ немесе аудармашы қолданыңыз.  
  3. Барлық скриншоттарды бір PDF құжатында жинаңыз.

> **Ескерту:** Тапсырмалар кітапта 323–374 беттер арасында орналасқан, exercise 13.1 – 13.8. Аударма қажет болса ИИ немесе аудармашы арқылы орындауға болады.

# Интерфейстегі типографика

## Теориялық бөлім
Типографика — мәтінді оқуға ыңғайлы етіп орналастыру өнері. Интерфейсте әріп өлшемі, қаріп түрі, жоларалық интервал, абзацтар мен маркерлер қолданушы тәжірибесін арттырады.

## Тапсырма
- **Файл:** `text.html` (студенттер өздері құрастырады)  
- **Кітап:** `Devid_Soyer_Makfarland_Novaya_bolshaya_kniga_CSS_proglib-142-210.pdf`  
- **Нұсқаулық:**  
  1. Кітаптағы соңғы беттердегі тапсырмаларды орындаңыз (бастапқы бет 198).  
  2. HTML файлында бірнеше абзац мәтін генерациялау.  
  3. CSS арқылы әр абзацқа стильдер беру (қаріп, өлшем, жоларалық интервал, маркерлер).  
  4. HTML файлының скриншоттарын жасап, PDF құжатында жинап тапсыру.

> **Ескерту:** Тапсырма кітаптың 198-бетінен басталып, 210-бетке дейінгі бөлімде орналасқан. (Файлдың 57-беті) Студенттер өздері `text.html` файлын құрастырады.
